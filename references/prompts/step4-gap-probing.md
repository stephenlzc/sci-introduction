# 第四步：双重视角缺口探测 Prompt 模板

---

## 输入

**第三步的对比分析**：

```
## 第三步：核心分歧诊断

### 批判性分析段落

[约200字的批判性分析]

### 张力根源分析

- 认识论层面的差异： [...]
- 制度/文化语境层面的差异： [...]
```

---

## Prompt

你是一位顶尖的国社科/自科基金撰写专家。

同时扫描国内外两个领域的盲区，找出能证明新研究具备绝对合理性的"研究缺口"。

请完成以下任务：

1. 指出国际前沿理论在解释本土语境时的"水土不服"——即国际研究**缺乏什么**
2. 指出国内经验研究在理论升华上的"短板"——即国内研究**缺乏什么**
3. **必须严格遵循此逻辑链**：
   > "国际研究缺乏**X**，而国内研究缺乏**Y**，因此，本研究填补了**Z**。"
4. 缺口必须是具体的、可操作的，指向一个质性过程或机制，而非泛泛的"研究不足"

---

## 输出格式

```
## 第四步：双重视角缺口探测

### 国际前沿的盲区（Gap 1）

[国际研究在解释本土语境时的水土不服：缺乏什么]

### 国内经验的短板（Gap 2）

[国内经验研究在理论升华上的缺陷：缺乏什么]

### 研究缺口定位

**逻辑链**：
- 国际研究缺乏 → [X]
- 国内研究缺乏 → [Y]
- 因此，本研究填补了 → [Z：一个具体的质性过程/机制]

**Prioritized Opportunity**：

[整合 X + Y，明确本文的优先切入点：用一句话精准陈述本文如何同时填补这两个缺口]

**本文切入点**：[用一句话精准陈述]
```
