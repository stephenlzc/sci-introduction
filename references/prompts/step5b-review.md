# 第五步 B：质量审查 Prompt 模板

---

## 输入

**初稿正文**：

```
## 国内外研究现状

[初稿正文]

---

## 参考文献

[1] ...
```

---

## Prompt

你是一位严谨的质检员。对照检查清单，逐项核查初稿质量。

**请逐项核查并输出**：

```
## 质量审查报告

### 1. 报菜名检查
- [ ] 是否存在"作者A发现X，作者B指出Y"的报菜名写法？
- [ ] 是否按主题聚类而非按作者组织？
问题（如有）：[具体位置和修改建议]

### 2. 废话套话检查
- [ ] 是否出现"值得注意的是"、"多姿多彩"、"随着...不断发展"等套话？
问题（如有）：[具体位置和修改建议]

### 3. 对比转折句检查
- [ ] 是否有至少2处对比性转折句式（"尽管...但..."、"然而...恰恰..."）？
- [ ] 转折是否服务于揭示国内外研究的根本分歧？
问题（如有）：[具体位置和修改建议]

### 4. 研究缺口逻辑链检查
- [ ] 是否明确指出国际研究缺乏什么（Gap 1）？
- [ ] 是否明确指出国内研究缺乏什么（Gap 2）？
- [ ] 是否清晰陈述本文填补了什么（Z）？
- [ ] 逻辑链：国际缺X → 国内缺Y → 本文填补Z 是否完整？
问题（如有）：[具体位置和修改建议]

### 5. 人称与语态检查
- [ ] 是否使用第三人称？
- [ ] 是否避免了"笔者认为"、"本文认为"等第一人称表达？
问题（如有）：[具体位置和修改建议]

### 6. 引用格式检查
- [ ] 引用是否为（作者, 年份）^{[1]} 双重格式？
- [ ] 是否有 (H1)^{[1]} 或 C1^{[1]} 等错误格式？
- [ ] 多个引用是否分开写（不是 [1][2] 连写）？
- [ ] 参考文献是否为单一顺序列表（不分中/英文节）？
- [ ] 引用格式是否符合 GB/T 7714-2015？
- [ ] 是否存在5+连续引用（串引）？
问题（如有）：[具体位置和修改建议]

### 7. 字数检查
- [ ] 字数是否在800-1200字范围内？
实际字数：XXX字

### 8. Evidence-bounding 检查
- [ ] 每个事实声明是否有文献或逻辑支撑？
- [ ] 是否存在过度推断？
问题（如有）：[具体位置和修改建议]

---

## 综合评分

| 检查项 | 通过 | 需修改 | 严重 |
|--------|------|--------|------|
| 报菜名检查 | ✓/✗ | - | - |
| 废话套话检查 | ✓/✗ | - | - |
| 对比转折句 | ✓/✗ | - | - |
| 缺口逻辑链 | ✓/✗ | - | - |
| 人称语态 | ✓/✗ | - | - |
| 引用格式 | ✓/✗ | - | - |
| 字数 | ✓/✗ | - | - |
| Evidence-bounding | ✓/✗ | - | - |

**总体评估**：[通过 / 需小修 / 需大改]
**修改建议**：[如有严重问题，列出具体修改建议]
```
