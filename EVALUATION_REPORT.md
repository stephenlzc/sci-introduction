## 六、核心发现

### Skill 的不可替代价值

| 能力 | With-Skill | Baseline |
|---|---|---|
| **话语体系分离** | 国际/国内矩阵清晰重构 | 混在"三节"结构里 |
| **消灭报菜名** | 3/3 通过 | 3/3 失败 |
| **批判深度** | 认识论层面张力分析 | 描述性罗列差异 |
| **缺口逻辑链** | 3/3 完整 | 3/3 模糊 |
| **Prioritized Opportunity** | 有 | 无 |
| **写作检查清单** | 内嵌自检 | 无 |

### Baseline 的系统性缺陷

1. **分节固化**：自动落入"国外研究→国内研究→比较评述"三段结构，形成国内外人为壁垒，无法有机整合。
2. **报菜名写法**：所有 baseline 输出均退化为"作者A指出…作者B发现…"的列举模式。
3. **缺口模糊**：所有 baseline 结尾的"研究不足"均为泛泛而谈，未遵循"国际缺X → 国内缺Y → 本文填补Z"的逻辑链。
4. **批判性弱**：缺乏认识论层面的张力分析，无法做到"制度环境作为存在性条件 vs 策略性资源"这类精准判断。
