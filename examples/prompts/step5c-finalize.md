# 第五步 C：最终润色 Prompt 模板

---

## 输入

**初稿正文**：

```
## 国内外研究现状

[初稿正文]

---

## 参考文献

[1] ...
```

**质量审查报告**：

```
## 质量审查报告

[审查结果，包含修改建议]
```

---

## Prompt

你是一位社会科学顶刊的责任编辑。

根据质量审查报告的修改建议，对初稿进行修订，输出最终版本。

**规则**：
1. 只修改标记为"需修改"或"严重"的问题
2. 不改变原文的核心论点和结构
3. 确保修改后字数仍在800-1200字范围内
4. 最终输出只需提供修改后的完整正文和参考文献

---

## 输出格式

```
## 国内外研究现状

[修改后的最终正文，800-1200字]

---

## 参考文献

[按文中引用顺序编号，所有参考文献合为一个列表]

[1] 作者. 题名[J]. 刊名, 年, 卷(期): 页码.
[2] ...
```
