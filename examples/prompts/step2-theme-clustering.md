# 第二步：主题聚类整合 Prompt 模板

---

## 输入

**第一步的对比矩阵**：

```
## 第一步：话语体系分类

### 对比矩阵

| 核心概念 | 国际话语包装 / 主导理论 | 国内话语包装 / 本土现实 |
|----------|------------------------|-------------------------|
| [概念X]  | [国际学者的理论化路径] | [国内学者的经验关切]    |

### 关键发现

- 国际研究的主导范式：[概括]
- 国内研究的本土特征：[概括]
```

---

## Prompt

你是一位精通扎根理论和主题聚类的资深质性研究员。

将第一步的分类打乱，按**主题**而不是**作者**来归纳文献，提炼背后的机制或质性体验——彻底消灭报菜名写法。

请完成以下任务：

1. 识别出 **3-4 个跨越国界的"核心主题"**（不是按作者分组，而是按概念维度）
2. 针对每个核心主题，提炼国内外学者的核心争论点
3. 聚焦于研究是如何捕捉**微观过程**和**真实处境**的，而不仅仅是罗列静态的变量结论

---

## 输出格式

```
## 第二步：主题聚类整合

### 核心主题一：[主题名称]
**概念维度**：X的生成机制 / Y的本土实践 / Z的制度约束

- **国际立场**：[概括核心主张，不提具体作者]
- **国内立场**：[概括核心主张，不提具体作者]
- **张力点**：[在这个维度上，双方最核心的分歧是什么]

### 核心主题二：[主题名称]
### 核心主题三：[主题名称]
（通常3-4个主题）

### 综合聚类概览

- 主题群A：[核心概念/方法类别]
- 主题群B：[核心概念/方法类别]
- 主题群C：[核心概念/方法类别]
```
