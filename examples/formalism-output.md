## 写作检查清单（自行复核）

- [ ] 没有"作者A提到X"的报菜名写法（应按主题聚类）
- [ ] 没有废话套话（重要值得注意的是等）
- [ ] 有明确的对比性转折句式（至少2处）
- [ ] 研究缺口逻辑链完整（国际缺X → 国内缺Y → 本文填补Z）
- [ ] 全文第三人称，学术声口冷峻
- [ ] 无过度推断，所有声明有文献/证据支撑
- [ ] 引用使用（作者, 年份）^{[1]} 双重格式，不是 (H1)^{[1]} 或 C1^{[1]}
- [ ] 多个引用分开写，不是 [1][2] 连写形式
- [ ] 参考文献合并为一个顺序列表，不按语言分节
- [ ] 引用格式符合 GB/T 7714-2015 标准
- [ ] 无5+串引
- [ ] 字数在800-1200字范围内
